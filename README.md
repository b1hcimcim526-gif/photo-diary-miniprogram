# plog · 照片日记

一个纯前端的照片日记网页应用，创作栏和日记本两个独立功能，数据保存在浏览器本地（localStorage），无需后端。

## 功能

### 创作栏

- 首页是按天分组的朋友圈式列表，今天没记录时显示一个"+"占位
- 点进某一天可以添加照片：单张、多张、拼图（6 种网格模板，支持拖动调整每格裁切位置）
- 照片默认按 3:4 画布显示，非 3:4 的图片自动补白边，点击照片可缩放/拖动调整裁切
- 每张照片可以加文字（可拖动位置、调整字号），文字会直接合成进照片
- 顶部日历图标可以跳转选择过去的日期记录

### 日记本

- 仅浏览和导出，不能编辑或删除
- 按月分组，每月照片均分成最多 9 份，拼成长图，横向可滑动浏览，像朋友圈相册一样
- 顶部"导出"按钮导出当前浏览到的月份的全部长图

## 使用

直接用浏览器打开 `index.html`，或用任意静态服务器托管：

```bash
npx serve .
```

## 部署

已连接 Vercel，push 到 `master` 分支会自动部署。
